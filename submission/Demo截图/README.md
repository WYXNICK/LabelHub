# Demo 截图说明

`submission/Demo截图/` 收录的是基于当前本地运行页面重新生成的核心截图，用于快速展示 LabelHub 的主要产品形态。截图通过 Chrome DevTools 对真实浏览器页面进行检查与截取生成，不使用早期原型图或开发过程中的遗留截图。

截图不依赖公网演示地址，避免演示环境数据变化影响材料稳定性；公网地址仅在演示环境说明中作为访问入口记录。

## 使用说明

- 截图仅用于说明关键页面布局和交互结构，实际演示以运行环境中的实时页面为准。
- 截图覆盖 Owner、Labeler、Reviewer 三类核心角色以及任务、模板、标注、AI 预审、人工审核和导出等关键链路。
- 后续如需更新截图，应优先使用本地或独立演示数据环境重新截取，避免直接依赖公网环境中的实时数据状态。
